
Data cleaning

1. handle Missing values
   > check which columns have Missing values (nulls)


Strategies to handle:
   > Drop Missing row/columns (only if very few )
    impute with :
                1. mean/median > for numerical data
                2. mode > for categorical data
                3. Advanced: linear regression, KNN , or interpolation (for future learning)

2. remove duplicates
   > detect and drop exact duplicates rows

3. Fix Data types
    > Convert Wrong types (eg. number stored as strings , dates as text)

4. handle inconsistent categories
    clean up categorical values like:
      >  "male" , "male", "MALE", > should all becomes "males" 

5. detect and handle outliers
    > use boxplots , IQR or Z-score
        handle by :
               > remove (if clearly Wrong)
               > capping (eg. to 95th percentile)

6. fix logic or Domain errors
   > Eg. age = -5 invalid , or BMI = 150 likely an error
   > can replace with mean, median or remove
